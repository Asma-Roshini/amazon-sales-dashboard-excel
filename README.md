# Amazon Sales Dashboard (Excel)

## Overview
This project analyzes Amazon product performance using Excel Pivot Tables and Dashboards.

## Dataset
- Source: Amazon Sales Dataset
- Records: 1,000+ products
- Metrics: Rating, Rating Count, Price, Discount

## KPIs
- Total Products
- Total Rating Count
- Average Rating
- Average Discount Percentage

## Dashboard Insights
- Identified brands with highest customer demand
- Analyzed category-wise engagement
- Compared price vs customer ratings
- Highlighted top 10 popular products

## Tools Used
- Microsoft Excel (Pivot Tables, Pivot Charts, Slicers)
- GitHub (Portfolio)
